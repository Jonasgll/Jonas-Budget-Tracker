# Budget Tracker — SimplovaStudio

Interaktiver Budget Tracker als Progressive Web App (PWA).

**Auf iPhone installieren:**
1. Link in Safari öffnen
2. Teilen-Button tippen ⎙
3. "Zum Home-Bildschirm" wählen
4. "Hinzufügen" tippen

**Auf Android installieren:**
1. Link in Chrome öffnen
2. "Installieren" Banner antippen

**Features:** Dashboard · Einnahmen · Ausgaben · Sparziele · Schulden · Jahresübersicht · Dark Mode · DE/EN
